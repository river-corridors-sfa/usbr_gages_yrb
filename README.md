# usbr_gages_yrb
Jupyter Notebook (Python) to download, analyze and plot U.S. Bureau of Reclamation stream gages in YRB

## Prerequisites

* Python3
* Numpy
* Matplotlib
* Pandas
* [tqdm]([https://pypi.org/project/tabulate/](https://github.com/tqdm/tqdm))
